sprint: An Agile Methodology

A sprint is a fixed-length time box, during which you work on a set of tasks to complete. Our Foundations work was set-out and delivered in this format. This mimics how it is in the Workplace, giving us a bit of insight into the Industry

Each week, we had new concepts of Programming Languages (peep the logos of the languages we engaged with each week) to learn and submit work from, as well as going through our all important Human Skills. Discover each sprint below and see what's been learnt!
